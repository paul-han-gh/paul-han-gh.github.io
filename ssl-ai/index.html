Individual Exercise

Conversation with AI

5 Minutes
Ask your LLM to help you identify any needed precautions—both short-term and long-term—when using AI.

Tips for the Exercise

1
For context, add a little profile about yourself ("I use AI for the following use cases...")
2
If you already use an LLM and you have the conversation feature enabled, ask it to summarize how you use AI today
3
Have a back-and-forth. Don't be satisfied with the first answer
4
When you reply in the back-and-forth, be open-ended ("Here's what I think... What do you think?")

Markdown Exercise

5 Minutes
Base Text
prompt instructions act as a professional nutritionist task create a 3-day meal plan goals high protein low carb no nuts ingredients to include chicken broccoli eggs spinach instructions provide a shopping list first then the meal plan breakdown for breakfast lunch and dinner finally include a section on storage tips important notes make sure all meals take less than 30 minutes to cook
Instructions
  • Use # or ## for section headers.
  • Use **bold** for specific constraints or "must-haves."
  • Use - or * for the list of ingredients.
  • Use 1., 2. for the sequence of instructions.

Individual Exercise

5 Min
1
Brainstorm a Tool
Think of a tool that does one thing well.
  • Translate a text from English to Korean
  • Create a memory game for a Bible verse
  • A simple budget calculator
2
The Prompt
Ask AI to make it, then copy/paste this exactly at the end of your prompt:
Please create me static webpage For icons, use icons from iconify.design Put CSS in style tag, JS in script tag (single file) Give me the result in a code block
The Workflow
3
Copy the output code into Notepad (Windows) or TextEdit (Mac).
4
Save the file as a .html file (e.g., game.html).
5
Open the file with your web browser.
6
Iterate: If it doesn't work, tell the LLM what's wrong and repeat steps 3-5.
Context Engineering Skill 01

The Socratic Method?

1
Assert a thesis
2
Find counter arguments or complete refutations
3
Start to believe in the counter arguments strongly
4
Find counter arguments to those counter arguments
5
Start to believe those now
Conclusion
You now believe that your thesis is both true and false

Individual Exercise

5 Min
Think of a high-level problem that you're dealing with right now.
1
Identify what you currently think is the best way to deal with the problem.
2
Ask AI to create a steelman argument as to why that's a horrible idea.
3
Then, ask AI to completely dissect and refute the argument that it just made.
4
Reflect on the tradeoffs that are part of the problem space.
