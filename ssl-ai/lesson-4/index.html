Background

What is Cowork and why does it matter?

Cowork is a research preview built into the Claude Desktop app that brings the same agentic capabilities behind Claude Code to knowledge workers — no terminal required. Instead of going back and forth in a chat window, you describe an outcome, step away, and come back to finished work: formatted documents, organized files, synthesized research, and more.

Think of the difference this way: Chat is asking a colleague a question. Cowork is delegating a project and checking back when it's done. Claude analyzes your request, breaks it into sub-tasks, coordinates parallel workstreams, and delivers outputs directly to your file system.

Step 01

Check your requirements

Before you begin, make sure you have everything you need.

Claude Desktop App

Cowork is built into the desktop app for macOS and Windows (x64). Not available on web or mobile.

Paid Claude Plan

Available on Pro ($20/mo), Max ($100–200/mo), Team, and Enterprise subscriptions.

Internet Connection

An active connection is required throughout every Cowork session.

Windows users: Cowork requires the latest version of Claude for Windows. Windows arm64 is not currently supported.

Step 02

Download Claude Desktop

If you don't already have the Claude Desktop app, head to claude.com/download and grab the installer for your operating system. On macOS you'll get a universal .dmg; on Windows, a standard x64 installer.

1

Visit claude.com/download

The site will auto-detect your OS. Click the primary download button.

2

Install the app

On macOS, drag the icon into Applications. On Windows, run the installer and follow the prompts.

3

Sign in to your account

Open Claude Desktop and log in with the account tied to your paid subscription.

Already have Claude Desktop? Make sure you're on the latest version — Cowork may not appear if your app is outdated. Check for updates in the app menu.

Step 03

Open Cowork & select a working folder

Once you're in Claude Desktop, you'll see a mode selector at the top of the window with tabs for Chat, Cowork, and Code. Click Cowork to switch into task mode.

Cowork will then ask you to select a local folder. This is the workspace Claude reads from and writes to — it can only access files in folders you explicitly grant. Start with a dedicated project folder to keep things clean while you learn.

1

Click the "Cowork" tab

The interface switches from Chat to the Tasks view.

2

Select a folder

Choose or create a folder like ~/cowork-workspace. Claude will read and write files here.

3

Choose your model

For best results, select Opus 4.6 with Extended Thinking enabled.

Security note: Cowork runs inside an isolated virtual machine on your computer. Claude can access the files you share, but requires explicit permission before permanently deleting anything.

Step 04

Run your first task

You're ready to delegate. Type a task description in the input field and hit Enter. Claude will analyze your request, propose a plan, and ask for your approval before executing. Start with something straightforward to build confidence.

/* Example first task prompt */

"Organize the files in this folder into subfolders
 by type: Documents, Images, Spreadsheets, and
 Other. Rename each file with today's date in
 YYYY-MM-DD format at the beginning."

Once Claude starts working, you'll see real-time progress indicators. You can follow along, steer mid-task, or step away entirely. For complex tasks, Claude spins up multiple sub-agents working in parallel.

Keep the app open. Claude Desktop must remain open for the entire duration of the task. Closing the app or putting your computer to sleep will end the session.

Step 05

Set up global & folder instructions

Cowork has no memory between sessions — every new task starts from scratch. That's by design, but it means you'll want to configure standing instructions so Claude already knows your preferences, tone, role, and output formats when each session begins.

Global instructions

Navigate to Settings → Cowork and click "Edit" next to Global instructions. Add details like your role, preferred output format, and tone of voice. These apply to every Cowork session.

Folder instructions

Add a markdown file (like INSTRUCTIONS.md) inside your working folder with project-specific context. Claude reads it automatically and can even update it during a session.

# Example: Global instructions

Role: Senior Product Manager at a B2B SaaS company
Tone: Professional but conversational, no jargon
Format: Always use bullet points for action items
Outputs: Prefer .docx for reports, .xlsx for data
Step 06

Connect your tools

Connectors let Cowork pull context from the services you already use — Slack, Google Drive, Notion, Figma, and dozens more. Once connected, Claude can search your Slack history, reference Google Docs, or pull data from Notion pages mid-task.

1

Open Settings → Connectors

Browse the connector directory inside Claude Desktop settings.

2

Add your services

Click "Add" on any connector you want. Authenticate with each service when prompted.

3

Reference them in tasks

Once connected, simply mention the tool in your prompt: "Search my Slack for the latest design review notes."

All connectors are free and included with your paid Claude subscription. No additional cost to enable any of them.

Step 07

Schedule recurring tasks

One of Cowork's most powerful features is the ability to set up tasks that run automatically on a cadence you define. Type /schedule in any task, or click "Scheduled" in the left sidebar to create and manage recurring automations.

Important: Scheduled tasks only run while your computer is awake and Claude Desktop is open. Tasks won't execute if the app is closed or your machine is sleeping.

Scheduling is ideal for recurring reports, daily file cleanup routines, periodic research digests, or any workflow you find yourself repeating. Define the task once and let Claude handle the rest.

What's Next

What you can build from here

Once your setup is complete, the real power of Cowork is in how you use it. Here are some of the most popular starting points.

File Organization

Sort hundreds of files into categorized folders, batch rename with consistent patterns, clean up downloads.

Spreadsheets & Data

Generate Excel files with working formulas, conditional formatting, and multiple tabs from raw data.

Presentations

Turn meeting transcripts, rough notes, or research into polished slide decks automatically.

Expense Reports

Drop receipt screenshots in a folder and get back a formatted expense spreadsheet with categories.

Research Synthesis

Combine web searches, articles, and notes into coherent, well-sourced reports and summaries.

Data Analysis

Outlier detection, cross-tabulation, time-series analysis, and visualizations on your data files.
