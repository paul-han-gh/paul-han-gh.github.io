AI Power User: Class Exercises

Three hands-on exercises that build on each other — from shaping AI behavior with roles, to creating custom assistants, to generating full interactive applications.

Exercise 1: Hidden Prompt Exercise 2: Build Your Expert Exercise 3: Interactive UI
How to use this page: Work through each exercise in order. You'll need access to at least one AI platform (ChatGPT, Claude, or Gemini). Each exercise builds on skills from the one before it.

Exercise 1: The Hidden Prompt

See how profoundly roles shape AI output

Put the "Hidden Prompt" concept into practice. By giving the AI a specific role upfront, you fundamentally change its tone, vocabulary, and perspective — even when the question stays exactly the same.

  1. Create 2 Different AI Chats

    Open your preferred AI tool (ChatGPT, Claude, or Gemini) in two separate tabs or windows.

  2. Assign Distinct Roles

    In the first message of each chat, give the AI a different "Role".

    Chat 1 — Role Prompt "You are an Economics Ph.D student."
    Chat 2 — Role Prompt "You are a 19th century theologian."

  3. Ask Identical Questions

    Provide the exact same series of questions in both of the active chats.

    Sample Question "Explain how the modern banking system works."

  4. Observe the Differences

    Compare the tone, vocabulary, formatting, and perspective of the resulting answers side-by-side. Notice how the same question produces completely different responses depending on the role.

What to look for: Pay attention to which answer uses jargon versus plain language, which one cites sources or scripture, and how the "frame" of the role changes what the AI considers important to mention.

Exercise 2: Build Your Own Expert

Create a reusable, pre-contextualized assistant

Create a permanent, pre-contextualized assistant to handle a specific, repetitive task: The ELI5 (Explain Like I'm 5) Translator.

  1. Open Your Platform

    Navigate to the creation screen for your platform. Follow the instructions for whichever tool you are using:

    ChatGPT — Custom GPT

    From the ChatGPT home screen, look at the left sidebar and click "Explore GPTs" (or go to chatgpt.com/gpts). In the top-right corner, click the "Create" button. This opens the GPT Builder. You'll see a "Configure" tab at the top — click it to access the instructions field where you'll paste the system prompt.

    Sidebar → Explore GPTs → Create → Configure tab
    Claude — Project

    From the Claude home screen, look at the left sidebar and click "Projects". Then click "Create Project". Give it a name (e.g., "ELI5 Translator"). Once inside the project, you'll see a "Project instructions" text area in the project settings panel on the right side — this is where you'll paste the system prompt.

    Sidebar → Projects → Create Project → Project instructions
    Gemini — Gem

    From the Gemini home screen, click "Gem manager" in the left sidebar (or go to gemini.google.com/gems). Click "New Gem" in the top-right. You'll see a text field labeled "Instructions" at the top of the page — this is where you'll paste the system prompt. Give your Gem a name before saving.

    Sidebar → Gem manager → New Gem → Instructions field

  2. Inject the System Prompt

    In the instructions field you just located, paste the following exact text:

    System Prompt "You explain complex concepts to a 5-year-old. Use simple words, fun analogies, and keep it under 3 sentences."

  3. Test Your Expert

    Save it, open a new chat with your newly created expert, and type a complex topic to test it.

    Topics to Try "Quantum Computing" • "The 2008 Financial Crisis" • "How DNA Works"
Going further: Try changing the system prompt to a different persona — a sports commentator, a pirate, a patient kindergarten teacher — and notice how the same topic gets explained in wildly different ways.

Exercise 3: Build an Interactive UI

Turn an AI prompt into a working web application

Expand our expert into a fully functioning application using Claude Artifacts, ChatGPT Canvas, or Gemini Gem Labs. You'll ask the AI to write code and render it as a live, interactive app — all within the chat window.

  1. Open the Workspace

    Each platform has its own name for the feature that renders live code. Follow the instructions for your tool:

    ChatGPT — Canvas

    Start a new chat in ChatGPT. Canvas activates automatically when you ask ChatGPT to write code or long-form content. You can also type "use canvas" in your prompt to explicitly open it. The Canvas panel will appear on the right side of the screen, showing live-rendered output alongside the conversation.

    New chat → Ask to write code (or type "use canvas")
    Claude — Artifacts

    Start a new chat on claude.ai. Artifacts are enabled by default — when you ask Claude to build something like a web app, it will automatically render the result in a panel on the right side of the screen. If you don't see it, go to Settings → Feature Preview and make sure "Artifacts" is toggled on.

    New chat → Ask to build an app → Artifact panel appears on right
    Gemini — Gem Labs

    From the Gemini home screen, open the left sidebar and click "Gem Labs". This is Gemini's space for building and running interactive code-based apps. Click "New" to create a new project, then paste your prompt in the chat field. The rendered app will appear in a preview panel alongside the conversation.

    Sidebar → Gem Labs → New → Paste prompt

  2. Prompt for an Application

    Paste the following exact text into your chat prompt:

    Application Prompt "Build a 1-page web app that explains 'The 2008 Financial Crisis'. Include 3 toggle buttons to change the reading complexity: ELI5, Proficient, and World-Class Expert."

  3. Interact with the App

    Watch the AI write the code and instantly render the app in the side panel (Canvas in ChatGPT, Artifacts in Claude, or the preview pane in Gem Labs). Click the different buttons to see the explanation adapt in real-time.

Try this next: Ask the AI to add features to the app you just built — for example, "Add a dark mode toggle" or "Add a fourth complexity level for a curious 10-year-old." This shows how AI-generated apps can be iterated on conversationally.
