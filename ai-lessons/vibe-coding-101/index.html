Module 01

Hugging Face
Spaces Connector

Connect Claude to over 500,000 AI models and Spaces on Hugging Face — directly from your Claude.ai chat window, no code required.

Requires a paid Claude account
01
Open Search & Tools
In any Claude chat, click Search and tools in the chat input box to open the connectors panel.
02
Add a Custom Connector
Select Add custom connector. HF is no longer in the built-in directory — you must enter the URL manually.
03
Enter the Remote MCP URL
Set the Remote MCP server URL to https://huggingface.co/mcp?login — this authenticates you via your HF account.
04
Add Image Generation Tools
Go to huggingface.co/settings/mcp and add Spaces tools such as mcp-tools/FLUX.1-Krea-dev for photorealistic images or mcp-tools/qwen-image for text-accurate images.
05
Generate Images in Claude
Ask Claude: "Use Krea to create an image of a vibrant garden with a Victorian house." Claude calls the HF Space and returns the generated image inline.

Module 02

Creating Single
Page Websites

Set up a Claude project with the right system prompt and you can generate fully self-contained HTML files — with CSS, JavaScript, icons, and AI images all in a single artifact.

Single HTML file CSS in <style> JS in <script> Iconify icons
Self-contained HTML
Everything lives in one file — no build step, no npm, no framework. Open it in any browser and it just works.
Iconify Icons
Add a CDN script tag for iconify-icon and use <iconify-icon icon="ph:…"> anywhere in your HTML.
Z-Image Turbo
Ask Claude to generate images via the HF Spaces connector. The tool returns a URL you can drop directly into your <img> tags.
Your Claude Project
System Prompt
Create a new Claude Project and paste this as the system prompt. Every conversation will produce a clean, self-contained HTML file ready to open in your browser.
System Prompt
You are a web designer. When building websites, always output a single complete HTML file with all CSS inside a <style> tag in <head> and all JavaScript inside a <script> tag before </body>.

For icons, use Iconify via CDN: <script src="https://code.iconify.design/iconify-icon/2.1.0/iconify-icon.min.js"></script> and reference icons as <iconify-icon icon="ph:name-bold"></iconify-icon>.

For images, use the Z-Image Turbo tool from the Hugging Face Spaces connector to generate real AI images, then place the returned URL in an <img> tag.

Never split your output across multiple files. The artifact must be complete and immediately openable in a browser.
Module 03

Using
Claude Code

Claude Code is a CLI tool that lets Claude read, write, and run code in your actual project — no copy-pasting required. It can browse files, run commands, and iterate directly on your codebase.

Full File System Access
Claude can read any file in your project, create new files, and edit existing ones with surgical precision.
Run Terminal Commands
Execute npm scripts, run tests, start servers, or any shell command — all from within the conversation.
Git-Aware
Claude understands your repo history, can create commits, and knows what has changed since the last commit.
MCP Tool Support
Connect MCP servers (like MotherDuck, Notion, or Hugging Face) for even deeper integrations right in your project.
Persistent Memory
Claude Code stores context in a CLAUDE.md file so it remembers your project conventions across sessions.

Module 04

Generate with
Taste-Skill

Taste-Skill is a design system for AI — a set of rules you give Claude to produce premium, visually sophisticated interfaces instead of generic templates. Three dials control the aesthetic output.

Design Variance
8 / 10
Asymmetric layouts, massive whitespace zones, masonry grids
Motion Intensity
6 / 10
Fluid CSS transitions, scroll reveals, spring animations
Visual Density
4 / 10
Standard app spacing, comfortable reading rhythm
Key Design Rules
Premium Typography
Use Geist, Outfit, Satoshi, or Cabinet Grotesk. Never Inter. Headlines at tracking-tighter with leading-none for impact.
Single Accent Color
Maximum 1 accent with saturation <80%. Absolute neutral base (Zinc/Slate). No neon glows or AI purple gradients.
No Centered Heroes
Variance >4 forces split-screen, left-aligned, or asymmetric layouts. Bento grids over 3-column equal-card layouts.
Perpetual Micro-Motion
Pulse, typewriter, float, shimmer effects keep components alive. Spring physics (stiffness 100, damping 20) for interactions.
No AI Clichés
Ban "Seamless", "Elevate", "Unleash" copy. No Unsplash photos. No John Doe / Sarah Chan. No 99.99% uptime metrics.
How to Apply
Copy SKILL.md into your project, then ask Claude to follow it. Or reference it inline: "Follow the taste-skill design rules at [URL]."
